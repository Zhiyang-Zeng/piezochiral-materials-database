# PiezoChiral Materials Database

An open database for exploring piezochiral materials through their crystallographic symmetries and chemical compositions.

## Features

* Search materials by **point group**, **space group**, or **chemical formula**
* Explore symmetry-allowed **piezochiral coupling forms**
* Browse classifications of piezochiral crystal classes
* Access materials predicted to exhibit piezochiral responses
* Visualize crystallographic symmetry information and tensor forms

## Data Source

Material entries are sourced by the Materials Project database. 

## Scientific Background

The database is based on the first theoretical framework establishing the piezochiral effect and the complete symmetry classification of piezochiral materials:

> Z. Zeng, M. Först, M. Fechner, X. Deng, A. Cavalleri, and P. G. Radaelli
> **The Piezochiral Effect**
> DOI: https://doi.org/10.48550/arXiv.2510.21674

This work derives the symmetry conditions under which mechanical strain can induce chirality and identifies the crystallographic point groups that support piezochiral coupling.
